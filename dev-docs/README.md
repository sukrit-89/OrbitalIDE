# Developer Documentation

This directory contains implementation and operational documentation for Orbital IDE.

## Documents

- `dev-docs/architecture.md`: System architecture and data flows.
- `dev-docs/features-status.md`: Implemented vs partial vs pending feature matrix.
- `dev-docs/backend-compiler.md`: Compiler backend internals and API contract.
- `dev-docs/local-development.md`: Local setup, test, and build workflow.
- `dev-docs/troubleshooting.md`: Known issues and practical fixes.

## Scope

These docs are intended for contributors and maintainers. End-user onboarding remains in `README.md`.
